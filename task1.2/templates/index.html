Хранилище ключ значение

Прототип API хранилища ключ-значение.

Получить все данные хранилища WEB:
http://127.0.0.1:3000/api/v1/storage/json/all

Получить все данные хранилища CURL:
curl -i -X GET http://127.0.0.1:3000/api/v1/storage/json/all

Получить данные хранилища по ключу WEB
http://127.0.0.1:3000/api/v1/storage/json/read?key=test

Получить данные хранилища по ключу CURL:
curl -i -GET http://127.0.0.1:3000/api/v1/storage/json/read?key=test

Добавить данные в хранилище WEB:
curl -i -H "Content-Type: application/json" -X POST -d '{"test1": "value2"}' http://127.0.0.1:3000/api/v1/storage/json/write
